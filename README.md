# ping pong
## contributors
 Emmanuel Mulatya
## specifications
* numbers divisible by three it output's ping.  
* numbers divisible by five it shows pong.  
* number divisible by both three and five it shows pingpong.  
## example
if you use the number 15 the output will be as follows:  
1  
2  
ping  
4  
pong  
ping  
7  
8  
ping  
pong  
11  
ping  
13  
14  
pingpong
